GIT TUTORIAL

1- Entre na pasta onde irá guardar o projeto:
	cd /caminho/para/a/pasta

2- Inicialize o git na pasta:
	git init

2- Crie um clone do repositório:
	git clone https://github.com/andrecaetanov/projeto1.git

3- Refaça os passos 1 e 2 dentro da pasta criada pelo comando clone.

4- Adicione o repositório remoto como origin:
	git remote add origin https://github.com/andrecaetanov/projeto1.git

5- Crie sua branch usando como o padrão seu primeiro nome começando com letra minúscula:
	git checkout -b primeironome

6- Adicione as alterações feitas:
	git add *

7- Cheque em qual branch está e quais alterações foram adicionadas:
	git status

8- Dê um commit com uma mensagem especificando as alterações realizadas:
	git commit -m "mensagem especificando o que foi feito"

9- Envie o commit feito para sua branch:
	git push origin suabranch

10- Volte para a master:
	git checkout master

11- Mescle a master com as alterações enviadas para sua branch (apenas quando solicitado pelo SM):
	git merge suabranch

12- Confirme o merge (apenas quando solicitado pelo SM):
	git push origin master

13- Puxe as alterações feitas pela equipe e atualize seu repositório:
	git pull
